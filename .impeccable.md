# Design Context — AMC IMAX 70mm Alerts

## Target Audience
Movie enthusiasts, specifically IMAX fans in NYC who want to catch Project Hail Mary in the premium 70mm format. Tech-savvy enough to subscribe to alerts. Age 18-45. Passionate about the cinematic experience — they care about format quality and will pay premium prices for the best presentation.

## Use Cases
1. Check if IMAX 70mm tickets are available for specific dates
2. Subscribe to get instant alerts when tickets drop
3. Compare showtimes across dates and formats
4. Quick-tap to buy tickets before they sell out
5. Share the alert page with friends planning to go together

## Brand Personality / Tone
**Cinematic. Urgent. Premium.**
- Feels like a movie event page, not a generic SaaS
- Urgency without anxiety — excitement about the experience
- Dark, theatrical atmosphere (like being in a cinema)
- Red accent inspired by theater curtains and AMC branding
- Clean, confident typography that doesn't compete with the content
- Motion should feel like film — smooth, deliberate, occasionally dramatic

## Aesthetic Direction
**Cinematic Noir meets Event Countdown**
- Deep blacks and warm charcoals (not blue-tinted)
- AMC red (#e50914) as the singular accent — used sparingly for maximum impact
- Large, confident type for movie title and dates
- Atmospheric texture: subtle film grain, light leaks, or projection artifacts
- Cards that feel like physical tickets or film strips
- Status indicators that feel alive — pulsing availability dots, animated countdowns
- Generous whitespace that lets content breathe like a movie poster

## Technical Constraints
- Next.js 15 + React 18 + TypeScript + Tailwind CSS v4
- Cloudflare Pages deployment (static export + edge functions)
- Must work beautifully on mobile (most users will check on phone)
- No heavy JS libraries for animations — CSS animations and transitions preferred
- Google Fonts only (loaded via next/font)
