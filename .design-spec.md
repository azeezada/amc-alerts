# Design Spec — AMC Theatres Website Reference

Match this spec exactly. Every decision should ask: "Does AMC's site do this?"

## Colors
| Token | Value | Usage |
|-------|-------|-------|
| --bg-base | #0A0A0F | Page background (near-black, not pure black) |
| --bg-surface | #1A1A1A | Card/elevated surfaces |
| --bg-elevated | #2A2A2A | Secondary cards, hover states |
| --border | #333333 | Borders, dividers |
| --border-subtle | #222222 | Subtle separators |
| --text-primary | #FFFFFF | Headings, primary text |
| --text-secondary | #CCCCCC | Body text, descriptions |
| --text-tertiary | #999999 | Metadata, captions |
| --text-disabled | #666666 | Disabled states |
| --accent | #E31837 | AMC Red — CTAs, brand elements |
| --accent-hover | #C02032 | Red hover state |
| --accent-dark | #9B1B2B | Red active/pressed |
| --gold | #F5C518 | Membership, ratings, premium badges |
| --link | #5BABF2 | Text links on dark backgrounds |
| --success | #4CAF50 | Confirmation only (NOT for availability) |
| --warning | #FF9800 | Limited availability, alerts |

### Rules
- NO green for availability status. Use white outline buttons (available) or gray (sold out)
- NO cream/beige/warm backgrounds. AMC is DARK
- NO purple-to-blue gradients
- Accent red used sparingly — only for CTAs and brand elements
- Tint all grays slightly warm (toward #1A1816 not #1A1A1A) for cinematic feel

## Movie Poster
- Use actual movie poster images (not emoji placeholders)
- Aspect ratio: 2:3 (standard poster)
- Hero usage: large, atmospheric, possibly with gradient overlay
- Thumbnail: can be circular crop (~70px) in list views

## Showtime Display Pattern
AMC groups showtimes as:
```
MOVIE TITLE
  └─ THEATER NAME
       └─ FORMAT (IMAX 70mm, Dolby Cinema, etc.)
            └─ [7:00 PM] [10:00 PM] [11:30 PM]  ← horizontal row of time pills
```

Showtime pills are white-outlined on dark, arranged horizontally.
On hover they invert (white fill, black text).
Sold out ones are gray with line-through.

## Format Badges
Don't use colored pills. Use small, clean text labels or brand-specific styling:
- IMAX: bold white text
- Dolby Cinema: can use Dolby's red-orange
- Premium: gold accent (#F5C518)
- Standard: default white

## What NOT to Do
- ❌ Cream/beige/warm backgrounds
- ❌ Serif fonts
- ❌ Green anything (green is NOT in AMC's palette)
- ❌ Film grain/film strip decorative elements (AMC's site is clean, not nostalgic)
- ❌ Countdown timers (AMC doesn't use them — keep if functional but restyle)
- ❌ "BEST" badges
- ❌ Dark mode toggle (site IS dark, no toggle)
- ❌ Decorative red bars at top
- ❌ Card borders with accent colors
- ❌ Emoji as placeholders (use real images or clean icons)
